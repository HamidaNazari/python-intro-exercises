# Countup & Countdown Project

**Author:** Hamida Nazari  
**Language:** Python 3  

---

## Overview
This project demonstrates recursion using two functions: `countup` and `countdown`.

- **Countup:** Recursively counts up from a negative number to 0.
- **Countdown:** Recursively counts down from a positive number to 0.
- Both print "Blastoff!" at the end.

---

```bash
python countup_countdown.py
